Enviado para coletar informações vitais, você vê sua equipe ser eliminada e fica sozinho em território inimigo. Agora, sua única missão é sobreviver, avançar e completar o objetivo custe o que custar.

Descrição Jogo

Slugbyte é um jogo de tiro 2D clássico onde ondas de inimigos avançam constantemente para eliminar o jogador. A cada horda derrotada, uma nova área é desbloqueada, trazendo desafios ainda mais intensos e inimigos mais perigosos.

Como instalar e jogar

1. Baixar o jogo

Baixe o arquivo .zip do jogo e extraia em uma pasta no seu computador.

2. Instalar o Python

  1. Acesse: python.org
  2. Baixe a versão mais recente
  3. IMPORTANTE: marque a opção "Add Python to PATH"
  4. Clique em "Install Now"

3. Verificar instalação

Abra o terminal (Prompt de Comando) e digite:

python --version

Se aparecer a versão, está tudo certo.

4. Instalar o Pygame

No terminal, digite:

pip install pygame

5. Rodar o jogo

  1. Abra o terminal na pasta do jogo
  2. Digite:
python jogo.py
